# 💬 Real-time Chat API (FastAPI + WebSocket)

A modern **real-time chat backend** built with **FastAPI**, **WebSockets**, **SQLAlchemy**, and **PostgreSQL**.  
This project is designed as a scalable backend for chat applications similar to Telegram / Discord.

---

## 🚧 Project Status

> ⚠️ This project is currently **under active development**  
Features and structure may change over time.

---

## ⚙️ Tech Stack

- FastAPI
- WebSockets
- SQLAlchemy (Async)
- PostgreSQL
- Redis
- JWT Authentication
- Uvicorn

---

## ✨ Features

- 🔐 JWT Authentication (Access & Refresh Tokens)
- 👤 User registration & login system
- 💬 Real-time chat using WebSockets
- 🧑‍🤝‍🧑 Private & group chats
- 📩 Message sending & receiving in real-time
- 🕓 Message history support
- 📦 Scalable project structure
- 🧠 Clean architecture (services / schemas / models separation)
- 🔄 Redis-ready for pub/sub scaling
- 🧩 Modular design for future microservices migration
